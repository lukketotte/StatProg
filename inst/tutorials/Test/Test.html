  1. Create a matrix xx with \(1000\) rows and \(2\) columns. You can use rnorm() to fill it with numbers. What happens if you use the variance function on xx? 
    variance <- function(x, unbiased = TRUE) {
  n <- length(x) # Number of observations
  if (unbiased) {
    res <- sum((x - mean(x))^2)/(n - 1) # Sample variance (unbiased)
  } else {
    res <- sum((x - mean(x))^2)/n # Sample variance (ML)
  }
  return(res)
}
